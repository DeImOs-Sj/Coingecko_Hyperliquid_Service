# Token Insight & Analytics API

## Overview
This backend service provides a comprehensive suite of tools for cryptocurrency analysis. It includes:
1. **Token Insight API**: Generates professional market insights using CoinGecko and Groq AI.
2. **HyperLiquid PnL API**: Calculates daily profit and loss for HyperLiquid wallets.

## Tech Stack
- **Framework**: [NestJS](https://nestjs.com/)
- **AI Engine**: [Groq](https://groq.com/) (using Llama-3.3-70b-versatile)
- **Market Data**: [CoinGecko API](https://www.coingecko.com/en/api)
- **Documentation**: [Swagger](https://swagger.io/)
- **Infrastructure**: [Docker](https://www.docker.com/) & Docker Compose

## Features
- **Token Analysis**: Automated insights generated by AI based on real-time market data.
- **PnL Tracking**: Breakdown of realized/unrealized PnL, fees, and funding on HyperLiquid.
- **Auto-Documentation**: Live Swagger UI at `/api/docs`.
- **Pagination**: Support for paginated results in daily PnL lists.
- **Validation**: Strict input validation using `class-validator`.

## Setup Instructions

### 1. Prerequisites
- [Node.js](https://nodejs.org/) (v20+)
- [Docker](https://www.docker.com/) (recommended)
- [Groq API Key](https://console.groq.com/)

### 2. Environment Configuration
Copy the `.env.example` file to `.env` and fill in your API key:
```bash
cp .env.example .env
```
Edit `.env`:
```text
GROQ_API_KEY=your_actual_key_here
PORT=3000
```

### 3. Running with Docker (Recommended)
```bash
docker-compose up --build
```
The app will be available at `http://localhost:3000`.

### 4. Running Locally
```bash
npm install
npm run start:dev
```

### 5. Accessing API Docs
Once running, visit:
`http://localhost:3000/api/docs`

## API Endpoints

### Token Insight
`POST /api/token/:id/insight`
- **Body**: `{ "vs_currency": "usd", "history_days": 30 }`
- **Description**: Fetches metadata and market history for a token and generates an AI insight.

### HyperLiquid PnL
`GET /api/hyperliquid/:wallet/pnl?start=YYYY-MM-DD&end=YYYY-MM-DD&page=1&limit=10`
- **Description**: Reconstructs daily PnL from trades and current positions.
